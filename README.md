likethemammal/react-primitives-touchable-bug
=========

Reproduction of a bug with Touchable in react-primitives

![Screenshot of the bug](https://i.imgur.com/4U5vlj6.png)

## Installation

```js
  //clone repo

  npm install
```

## Development

```js
  //in that same root dir
  npm run build
  
  //after the build
  cd example/
  
  npm run dev
  
  //go to http://localhost:8080/
```
